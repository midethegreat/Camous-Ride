# Vendor Dashboard

A comprehensive vendor management dashboard for the campus ride platform.

## Features

### Core Vendor Features

- Vendor onboarding
- Product/menu management (add/edit/delete items)
- Incoming orders dashboard
- Order status updates (accepted, preparing, delivered)
- Notifications (WhatsApp-style alerts UI)
- Sales/earnings overview

### Additional Features

- Chat/messaging UI (WhatsApp-style chat)
- Push notification UI components
- Admin-like overview screen
- Multi-dashboard switching

### Integrations (UI Only)

- WhatsApp notification concept
- Payment UI (card, wallet, crypto)
- Map interface placeholders (Google Maps style)

## Design

- Tech-savvy, minimal, clean interface
- Matches existing app theme
- Uses icons, cards, and modern layout spacing
